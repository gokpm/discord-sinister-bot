# discord-sinister-bot

### 1. Pre-Requisites:
1. Python 3 or higher
2. Discord bot token
3. Install the required modules from the requirements.txt file

### 2. Setup:
1. Create a .env file.
2. Open .env file and update the following:
  1.BOT_TOKEN
  2.REDDIT_CLIENT_ID
  3.REDDIT_CLIENT_SECRET
  4.REDDIT_USER_AGENT
3. Run sinister_bot.py
